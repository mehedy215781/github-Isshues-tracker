1️⃣ What is the difference between var, let, and const?
Ans:-
var: It is function-scoped and allows you to both change its value and redeclare the same name, which can cause bugs.
let: It is block-scoped, meaning it only works inside the brackets where it is write;we can change its value but cannot redeclare it.
const: It is also block-scoped, but once we set a value, it can never be changed or redeclared.
Hoisting: If we use var before declaring it, it shows undefined, but let and const will give an error.

2️⃣ What is the spread operator (...)?
Ans :-The Spread Operator (...) is a powerful tool in JavaScript that allows we to "spread" or expand the elements of an array or the properties of an object into another place.

3️⃣ What is the difference between map(), filter(), and forEach()?
Ans:-
 map() - 
 *It creates a new array by changing each element of an array. The original array remains unchanged.
 *When we want to modify all the data in an array 
 filter()-
 *It checks a specific condition and creates a new array with the items that meet that condition.
 *When you want to find specific data from a large list
 forEach()  -
 *It generally works like a loop and 
 *It is used to perform an operation on each item in the array But it does not return a new array on its own.
